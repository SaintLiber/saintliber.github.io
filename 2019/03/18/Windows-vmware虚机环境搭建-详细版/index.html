Windows-vmware虚机环境搭建-详细版

简介

虚拟机是所有环境的基础，所以先弄一个虚拟机的傻瓜文档作为开篇。

这里假设各位看官已经安装好了vmware workstation:

和Xshell:

相关的安装软件和方法就自行百度咯

vmware workstation虚拟网络编辑
这里vmware的虚拟网络有三种模式

  1. 桥接
  2. NAT
  3. HOST-Only

这里我们直接选择最简单的NAT模式，让虚拟机通过宿主机的网络连接外网，外网却无法连接到虚拟机，相当于隔了一层防火墙，这里我们设置了一个网段，让虚拟机的IP地址都通过DHCP服务从这个网段中按照顺序自动获得IP地址，如下我们配置了88，是个很吉利的网段，并且我们设置了IP的有效时间为俩月应该够我们折腾一段时间了。
